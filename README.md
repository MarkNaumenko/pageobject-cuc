# pageobject-cuc

To run this project, first write 'npm install -g gulp' in console, then
'npm install'. After this you can run test with command:
'gulp test --browser=firefox --view=desktop --tags=@notimportant'
for exapmle.
You can use 
	browser parameters:
	- firefox
	- chrome
	view parameters:
	- desktop
	- mobile
	tags:
	- @notimportant
	- @important
